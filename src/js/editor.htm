Создание нового блока слайдов

Редактирование блока слайдов

Допустимы символы [a-zA-Z_-]

Если 0, то ширина auto

Если 0, то высота auto

Переменные шаблоны: {v#url} - ссылка, {v#filehash} - идентификатор картинки, {v#title} - заголовок, {v#imgwidth} - ширина картинки, {v#imgheight} - высота картинки, {v#code} - дополнительный HTML код.

Если задействован персональный шаблон, то все слайды, при сборке, будут использовать его.
